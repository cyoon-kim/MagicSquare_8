# Report · Transcript Export

Report 폴더에 보고서 생성하고, Prompt 폴더에 Transcript도 Export 해줘.

**필수:** 아래 skill을 먼저 읽고 그대로 실행한다.

1. `.cursor/skills/export-report-transcript/SKILL.md` — 체크리스트·번호·파일명 규칙
2. `.cursor/skills/export-report-transcript/templates.md` — Report/Transcript 템플릿

**원칙**

- Report + Prompt **쌍**만 생성 (하나만 X)
- `Report/` 최대 NN + 1 사용 (현재 09 → 다음 10)
- Prompt 접미사: `_transcript.md`
- Report ↔ Transcript 상호 링크
- Git commit/push는 요청 없으면 하지 않음

완료 후 생성 경로·번호를 표로 보고한다.
